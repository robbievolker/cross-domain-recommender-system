About this website


This web application is designed to provide you with books, films and games recommendations based on an input title. Feel free to contribute via creating an account in order to add new items to the database, upvote existing tags on items and contribute to a dataset that can be used by future developers and researchers to create more advanced recommender systems for users.


The core of this application's functionality is the recommender system. For transparency, this system works by:

1) Using the Levenshtein distance to detect title similarity: This accounts for 20% of the weighting.

2) Performing a Cosine similarity rating on tags associated with two items: This accounts for 60% of the weighting.

3) Using spacy to conduct an entity analysis on the titles: This accounts for 20% of the weighting.


With thanks to the following API providers: Google Books, OMDb and IGDb.
